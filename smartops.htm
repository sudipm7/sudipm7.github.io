Operation

Resolves the exact host/network object for this IP (show hosts / show networks, exact match on ipv4-address), then runs where-used uid <uid> — the same calls used by decommision_object_v10.sh.
Adds the new object to the same groups and to source/destination of the same access rules as the reference object. Direct NAT and Threat Prevention references are report-only — same behavior as replication_script_v2.sh — since NAT should generally be covered by group membership instead of direct edits.
